1. JavaScript란 무엇인가?

- 웹 페이지의 내용을 정의하는 HTML

- 웹 페이지의 레이아웃을 지정하는 CSS

- 웹 페이지의 동작을 프로그래밍하는 JavaScript

2. JavaScript의 기능

- JavaScript에서 HTML 속성 값을 변경할 수 있다.

- JavaScript로 HTML 컨텐츠를 변경할 수 있다.

- JavaScript로 HTML 스타일, CSS를 변경할 수 있다.

- JavaScript에서 HTML 요소를 숨길 수 있다.

- JavaScript로 HTML 요소를 표시할 수 있다.

3. JavaScript의 태그

- HTML에서 JavaScript 코드는 script와 /script 태그 사이에 삽입된다.

- JavaScript 함수는 JavaScript 코드의 블록으로, 호출 시 실행할 수 있다.

- HTML 파일에 원하는 만큼 script를 배치할 수 있다. script는 HTML 파일의 body, head 섹션에 모두 배치할 수 있다.

4. JavaScript의 특징

- JavaScript 프로그램은 컴퓨터가 실행할 명령어 목록이다. 프로그래밍 언어에서 이러한 명령어를 statement라고 한다. 즉 JavaScript 프로그램은 이런 statement의 목록이다.

- JavaScript statement는 값, 연산자, 식, 키워드, 주석으로 구성된다.
