1. jQuery는 무엇인가?

- jQuery는 JavaScript의 라이브러리이다.

- jQuery는 JavaScript 프로그래밍을 단순화할 수 있고, 쉽게 사용할 수 있게 한다.

- jQuery를 사용해서 많은 줄의 JavaScript 코드를 간단한 코드로 변경할 수 있고, 복잡한 코드를 단순하게 만들 수 있다.

- jQuery는 HTML/DOM 조작, CSS조작, HTML 이벤트 메서드, 효과 및 애니메이션 등이 있다.

2. jQuery의 구문

- jQuery Syntax는 HTML 요소를 선택하고, 요소에 대한 일부 작업을 수행한다.

- 기본 구문은 $(selector).action()이다.

- jQuery를 정의하고 접근하기 위한 ‘$’, HTML 요소를 선택하기 위한 selector, 요소에 대해 수행할 action()이다.

3. jQuery의 selector

- jQuery selector를 사용하면 HTML 요소를 선택하고 조작할 수 있다.

- jQuery selector는 이름, ID, 클래스, 유형, 속성, 속성 값 등을 기반으로 HTML 요소를 찾거나 선택하는 것에 사용된다. 기존 CSS selector를 기반으로 하며, 또한 일부 자체 사용자 지정 selector가 있다.

- jQuery selector는 ‘$’ 기호와 괄호로 시작한다.

4. jQuery의 Event

- 웹 페이지가 응답할 수 있는 모든 수행을 이벤트라고 한다. 이벤트는 어떤 일이 일어나는 정확한 순간을 나타낸다.

- 예를 들면 요소 위로 마우스 이동, 라디오 버튼 선택, 요소 클릭 등이 있다.

- ‘fires/fired’라는 용어는 이벤트와 함께 자주 사용된다. 예를 들면 ‘The keypress event is fired, the moment you press a key’처럼 사용된다.

5. 자주 사용되는 jQuery 이벤트 메소드

- $(document).ready()를 사용하면 문서가 완전히 로드되었을 때 함수를 실행할 수 있다.

- Click()은 이벤트 핸들러 함수를 HTML 요소에 첨부한다 이 메소드는 사용자가 HTML 요소를 클릭할 때 실행된다.
