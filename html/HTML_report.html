1. HTML은 무엇인가?

- HTML은 hyper text markup language를 의미한다.

- 웹 페이지를 만들기 위한 표준 markup language이다.

- 웹 페이지의 구조를 설명한다.

- HTML은 여러 요소들로 구성되어 있다. 그 요소들로 내용을 표현하는 방법을 알 수 있다.

2. HTML의 사용

- 모든 HTML 파일은 “DOCTYPE html”형식의 문서 유형의 선언으로 시작한다.

- HTML 파일은 html,/html 사이에 작성한다.

- HTML 파일에서 실제로 보이는 부분은 body,/body 사이에 작성한다.

3. HTML의 특성

- 모든 HTML 요소는 속성을 가질 수 있다.

- 특성은 요소에 대한 추가적인 정보를 제공한다.

- 속성은 항상 시작 tag에 지정된다.

- 속성은 일반적으로 이름과 값으로 구성되어 있다.

4. CSS는 무엇인가?

- CSS는 웹 페이지의 레이아웃을 형성하는 것에 사용된다.

- CSS를 사용하면 색상, 글꼴, 텍스트 크기, 요소 사이의 간격, 요소 배치 및 배치 방법, 사용할 배경 이미지 또는 배경 색상, 다양한 장치와 화면 크기에 대한 다양한 디스플레이를 제어할 수 있다.

5. CSS의 사용

- CSS는 3가지 방법으로 HTML 파일에 추가할 수 있다.

- Inline : HTML 요소 내부의 스타일 속성을 사용한다.

- Internal : head 섹션의 style를 사용한다.

- External : 외부 CSS파일에 링크하기 위해 link를 사용한다.

6. JavaScript는 무엇인가?

- HTML의 script 태그는 클라이언트의 스크립트를 정의하는 것에 사용된다.

- script 요소는 스크립트 파일을 포함하거나 src 특성을 사용해서 외부 스크립트 파일을 가리킬 수 있다.

- 스크립트의 일반적인 용도는 이미지 조작, 양식 변환, 콘텐츠의 변경이다.
