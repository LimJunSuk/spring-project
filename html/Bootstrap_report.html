1. Bootstrap은 무엇인가?

- Bootstrap은 빠르고 쉬운 웹 페이지 개발을 위한 front-end framework이다.

- Bootstrap은 타이포그래피, 폼, 버튼, 테이블, 네비게이션, 모달, 이미지 카루셀 등 다양한 것을 위한 HTML, CSS 기반의 디자인 템플릿과 추가적인 JavaScript 플러그인을 포함한다.

- Bootstrap은 반응성이 뛰어난 디자인을 쉽게 만들 수 있는 기능을 제공한다.

2. Bootstrap의 장점

- 사용 편의성 : HTML과 CSS에 대한 기본적인 지식만 있으면 누구나 Bootstrap를 쉽게 사용할 수 있다.

- 반응성 기능 : Bootstrap의 반응성 CSS는 전화기, 태블릿 및 데스크톱에 맞게 조정된다.

- 모바일 우선 접근 방식 : Bootstrap에서 모바일 우선 스타일은 핵심 framework의 일부이다.

- 브라우저 호환성 : Bootstrap version 5는 모든 최신 브라우저(크롬, 파이어폭스, 엣지, 사파리, 오페라)와 호환된다. IE11 및 다운에 대한 지원이 필요하면 Bootstrap version 3 또는 4를 사용해야 한다.

3. Bootstrap의 Container

- Bootstrap의 Container에서는 컨테이너 내부의 내용을 패딩하기 위해 사용한다. 두가지 container 클래스를 사용한다.

- .container 클래스는 반응성 고정 너비 container를 제공한다.

- .container 클래스를 사용하여 반응성 고정 너비 container를 만들 수 있는데 이는 화면 크기에 따라 최대 너비가 달라진다.

- .container-fluid 클래스는 viewport의 전체 너비를 포함하는 전체 너비 container를 제공한다.

- .container-fluid 클래스를 사용하여 항상 화면의 전체 너비에 걸쳐 있는 전체 너비 container를 만들 수 있다. 즉 너비는 항상 100%이다.

4. Bootstrap의 Grid system

- Bootstrap의 Grid system은 flexbox로 구축되어 있으며 페이지 전체에 최대 12개의 열을 만들 수 있다.

- 12개의 열을 모두 개별적으로 사용하지 않을 때는 열을 함께 그룹화하여 더 넓은 열을 만들 수 있다.

- Bootstrap의 Grid system은 반응성이며, 화면 크기에 따라 열이 자동으로 다시 정렬된다.

- 사용되는 열의 총 개수가 12개 이하인지 확인해야 한다. 사용 가능한 12개 열을 모두 사용할 필요는 없다.
