1. CSS는 무엇인가?

- CSS는 계단식 스타일 시트를 의미한다.

- CSS는 HTML의 요소가 화면에 표시되는 방법을 설명한다.

- CSS를 사용해서 웹 페이지의 레이아웃을 한번에 제어할 수 있다.

- 외부 스타일 시트는 CSS 파일에 저장된다.

2. CSS를 사용하는 이유

- CSS는 다양한 장치와 화면 크기에 대한 디자인, 레이아웃 및 디스플레이 변형을 포함하여 웹 페이지의 스타일을 정의하는 것에 사용된다.

3. CSS Syntax

- Selector는 스타일을 지정할 HTML 요소를 가리킨다.

- 선언 블록에는 ‘;’으로 구분된 하나 이상의 선언이 포함되어 있다.

- 각 선언은 ‘:’으로 구분된 CSS 속성 이름과 값을 포함한다

- 여러 CSS 선언은 ‘;’으로 구분되며 선언 블록은 괄호로 구분된다.

4. CSS의 종류

- External CSS : 외부 스타일 시트를 사용하면 파일 하나만 변경해도 전체 웹 페이지의 모양을 변경할 수 있다. 각 HTML 페이지는 link 요소, 헤드 섹션 내부의 외부 스타일 시트 파일에 대한 참조를 포함해야 한다.

- Internal CSS : 하나의 HTML 페이지가 고유한 스타일을 가질 경우 내부 스타일 시트를 사용할 수 있다. 내부 스타일은 style 요소 내부, 헤드 섹션 내부에 정의된다.

- Inline CSS : 단일 요소에 대해 고유한 스타일을 적용하기 위해 Inline 스타일을 사용할 수 있다. Inline 스타일을 사용하려면 스타일 속성을 관련 요소에 추가해야 한다. 스타일 속성은 모든 CSS 속성을 포함할 수 있다.

5. CSS의 요소

- Color : 색상은 미리 정의된 색상 이름 또는 RGB, HEX, HSL, RGBA, HSLA 값을 사용하여 지정한다.

- Background : 배경 속성은 요소의 배경 효과를 추가하는 것에 사용된다.

- Border : 테두리 속성을 사용하면 요소 테두리의 스타일, 너비 및 색상을 지정할 수 있다.

- Margin : 여백은 정의된 테두리 외부에 있는 요소 주위에 공간을 만드는 것에 사용된다.

- Padding : 패딩은 정의된 테두리 안에서 요소의 내용 주위에 공간을 만드는 것에 사용된다.
